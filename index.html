Emoji - A captcha bot

A Telegram group bot to manage the group to prevent the entry of suspicious and malicious people in order to harm the group by actions such as: spam, scam, ads, etc. (CAPTCHA with Emoji)

github address

This robot uses the PIL library to process images and inserts emojis into a jpg image and displays other emojis completely randomly in the form of a glass button, and the user must find and select the right emoji from the image. slow to release accesses such as writing, sending voice and sending media. If the user does not answer, he will be expelled from the group.
